# **MyOwnBrowser**
- A simple and basic Web Browser built with Python
# Getting Started
### Installation
After downloading or cloning the repo, Navigate to the directory containing the files and run
```pip3 install requirements.txt```
# How to run
From the project directory **run**
##### ``` python3 main.py```
